# JuniorCSharpTestTask
### Допущения при парсинге логов (Задача 3)

- Формат логов считается строго соответствующим примерам из задания.
- При разборе строк используется метод `String.Split`.
  - Для **Формата 1** разделителем является пробел `' '`. Сообщение восстанавливается объединением оставшихся частей, поэтому множественные пробелы внутри сообщения могут быть нормализованы до одного.
  - Для **Формата 2** разделителем является вертикальная черта `'|'`. Парсер предполагает, что символ `'|'` **не встречается** внутри поля "Сообщение".
